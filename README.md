# dict

Dicionarios para goldendict

# Instalação do goldendict 

#Debian: 

sudo dpkg -i goldendict_1.5.0_git20160508.g92b5485-1.1+b1_amd64.deb

OU

sudo apt install ./goldendict_1.5.0_git20160508.g92b5485-1.1+b1_amd64.deb

#Adicionar Dicionarios

abra o goldendict, vá na aba Editar > Dicionarios
Adicione a pasta "goldendict dic", atualize, aplique. Enjoy!

#Usando 

selecione uma palavra em inglês e precione Ctrl+c+c
